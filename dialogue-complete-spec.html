00 Vision & Automated Flow

Every Tuesday at 06:00 SAST, the system automatically fetches upcoming rugby fixtures, scores them for SA audience relevance, selects the best 3–5, generates a Kwagga/Marcus dialogue roundup, and saves it as a draft. The editor reviews and promotes it to the homepage. No manual fixture entry needed.

TUESDAY 06:00 SAST — AUTOMATED ↓ Cron fires → fetchUpcomingFixtures(rugby_union, 7 days) Calls API-Sports for URC, Currie Cup, Rugby Championship fixtures ↓ scoreAndSelectFixtures() → top 3–5 by SA relevance score SA teams +50, URC +40, derby +25, weekend +20, Currie Cup +30 ↓ TitleQueue.create({ title, fixtures, is_weekly_roundup: true }) ↓ triggerGeneration() → Claude generates full dialogue roundup Both persona system prompts active, [FIXTURE:][KWAGGA][MARCUS] delimiters ↓ Post saved as DRAFT with fixture_dialogues, article_format: weekly-roundup MANUAL — EDITOR (anytime Tuesday–Thursday) ↓ Editor reviews draft in arclink admin Check fixture selection, review dialogue quality, edit blocks if needed ↓ Editor publishes + clicks "Set as homepage" ↓ sarugbybets.co.za homepage shows new weekly dialogue SA rugby bettors read Thursday–Sunday before placing weekend bets
Tuesday 06:00 is intentional. Weekend fixtures are settled. Next weekend's fixtures are confirmed and being priced. Editors have Tuesday–Thursday to review. Bettors read Thursday–Friday. The cron runs at 04:00 UTC which is 06:00 SAST (Africa/Johannesburg, UTC+2).
01 API-Sports Fixture Fetcher
new servicearclink blog
Sign up first. Go to api-sports.io, create a free account (100 req/day), get your API key. Then call GET https://v1.rugby.api-sports.io/leagues with your key to confirm the exact league IDs for URC, Currie Cup, and Rugby Championship before hardcoding them. 
arclink/services/blog/src/services/api-sports.ts
import axios from 'axios';

const BASE = 'https://v1.rugby.api-sports.io';
const KEY = process.env['API_SPORTS_KEY']!;

// Confirm these IDs by calling GET /leagues first
const LEAGUE_IDS: Record<string, number[]> = {
  rugby_union: [
    61,   // URC — VERIFY
    116,  // Currie Cup — VERIFY
    58,   // Rugby Championship — VERIFY
    48,   // Super Rugby Pacific — VERIFY
  ]
};

export interface ApiSportsFixture {
  id: number;
  homeTeam: string;
  awayTeam: string;
  competition: string;
  venue: string;
  kickoff: string;   // ISO string
  matchLabel: string; // e.g. "Bulls vs Sharks · Loftus · Sat 17:00"
}

export async function fetchUpcomingFixtures(
  sport: string,
  daysAhead: number
): Promise<ApiSportsFixture[]> {

  const leagueIds = LEAGUE_IDS[sport] ?? [];
  const fixtures: ApiSportsFixture[] = [];
  const season = new Date().getFullYear();

  for (const leagueId of leagueIds) {
    try {
      const { data } = await axios.get(`${BASE}/games`, {
        headers: { 'x-apisports-key': KEY },
        params: { league: leagueId, season, next: daysAhead }
      });
      fixtures.push(...(data.response ?? []).map(mapFixture));
    } catch (err) {
      console.error(`[API-Sports] League ${leagueId} failed:`, err);
    }
  }

  return fixtures;
}

function mapFixture(g: any): ApiSportsFixture {
  const kickoff = new Date(g.date);
  const day = kickoff.toLocaleDateString('en-ZA', {
    weekday: 'short', timeZone: 'Africa/Johannesburg'
  });
  const time = kickoff.toLocaleTimeString('en-ZA', {
    hour: '2-digit', minute: '2-digit', timeZone: 'Africa/Johannesburg'
  });
  return {
    id: g.id,
    homeTeam: g.teams.home.name,
    awayTeam: g.teams.away.name,
    competition: g.league.name,
    venue: g.venue?.name ?? 'TBC',
    kickoff: g.date,
    matchLabel: `${g.teams.home.name} vs ${g.teams.away.name} · ${g.venue?.name ?? 'TBC'} · ${day} ${time}`
  };
}
02 Fixture Scoring & Selection
new servicearclink blog

Scores each fixture for SA audience relevance. SA teams and derbies score highest. Competitions ranked by local interest. Weekend timing boosted because SA bettors bet on weekends.

arclink/services/blog/src/services/fixture-selector.ts
import { ApiSportsFixture } from './api-sports';

const SA_TEAMS = [
  'Bulls', 'Stormers', 'Lions', 'Sharks', 'Cheetahs',
  'South Africa', 'Springboks'
];

const COMPETITION_SCORES: Record<string, number> = {
  'URC': 40, 'United Rugby Championship': 40,
  'Rugby Championship': 35,
  'Currie Cup': 30,
  'Super Rugby Pacific': 20,
  'Six Nations': 15,
  'Premiership Rugby': 10,
};

const SA_DERBIES = [
  ['Bulls', 'Lions'], ['Bulls', 'Sharks'],
  ['Stormers', 'Sharks'], ['Stormers', 'Lions'],
  ['Lions', 'Sharks'], ['Bulls', 'Stormers'],
];

export function scoreAndSelectFixtures(
  fixtures: ApiSportsFixture[],
  max = 5
): ApiSportsFixture[] {

  const scored = fixtures.map(f => {
    let score = 0;
    const home = f.homeTeam;
    const away = f.awayTeam;

    // SA team involvement
    if (SA_TEAMS.some(t => home.includes(t) || away.includes(t)))
      score += 50;

    // Competition tier
    for (const [comp, pts] of Object.entries(COMPETITION_SCORES)) {
      if (f.competition.includes(comp)) { score += pts; break; }
    }

    // Weekend timing
    const day = new Date(f.kickoff).getDay();
    if (day === 6 || day === 0) score += 20;
    if (day === 5) score += 10;

    // SA derby bonus
    if (SA_DERBIES.some(([a, b]) =>
      (home.includes(a) && away.includes(b)) ||
      (home.includes(b) && away.includes(a))
    )) score += 25;

    return { fixture: f, score };
  });

  return scored
    .sort((a, b) => b.score - a.score)
    .slice(0, max)
    .map(s => s.fixture);
}

cron.schedule('0 4 * * 2', async () => {
  console.log('[Weekly Roundup] Cron fired');

  const tenants = await BlogTenant.find({
    active: true,
    sport_key: { $exists: true }
  });

  for (const tenant of tenants) {
    try {
      await runWeeklyRoundup(tenant);
    } catch (err) {
      console.error(`[Weekly Roundup] Failed for ${tenant.name}:`, err);
    }

export async function runWeeklyRoundup(tenant: IBlogTenant) {
  // 1. Fetch fixtures
  const raw = await fetchUpcomingFixtures(tenant.sport_key, 7);
  if (!raw.length) {
    console.log(`[Weekly Roundup] No fixtures for ${tenant.name} — skipping`);
    return;
  }

  // 2. Score and select
  const selected = scoreAndSelectFixtures(raw);

  // 3. Build title
  const date = new Date().toLocaleDateString('en-ZA', {
    day: 'numeric', month: 'long', year: 'numeric'
  });
  const title = `${tenant.sport_label} Weekend Preview — ${date}`;

  // 4. Generate directly (skip queue for automation)
  const generated = await generatePost(tenant, title, undefined, selected);
  const parsed = parseWeeklyRoundup(generated.content);

  // 5. Save as draft
  await Post.create({
    tenant_id: tenant.id,
    title,
    slug: generated.slug,
    content: generated.content,
    excerpt: generated.excerpt,
    seo_title: generated.seo_title,
    seo_description: generated.seo_description,
    tags: generated.tags,
    categories: generated.categories,
    article_format: 'weekly-roundup',
    dialogue_blocks: parsed.fixtures.flatMap(f => f.blocks),
    fixture_dialogues: parsed.fixtures,
    fixture_list: selected,
    featured: false,
    status: 'draft',   // always draft — editor promotes
    generated: true,
    reading_time: generated.reading_time,
    word_count: generated.word_count,
    created_at: new Date(),
  });

  console.log(`[Weekly Roundup] Draft saved: "${title}" (${selected.length} fixtures)`);
}

07 Dialogue Parser — New Utility
new filearclink blog
arclink/services/blog/src/utils/dialogue-parser.ts
export interface DialogueBlock { persona: string; content: string; order: number; }
export interface FixtureDialogue { matchLabel: string; blocks: DialogueBlock[]; }
export interface ParsedWeeklyRoundup { fixtures: FixtureDialogue[]; isValid: boolean; error?: string; }

export function parseDialogueContent(raw: string): { blocks: DialogueBlock[]; isValid: boolean; error?: string } {
  const re = /\[(KWAGGA|MARCUS)\]([\s\S]*?)\[\/(KWAGGA|MARCUS)\]/gi;
  const blocks: DialogueBlock[] = [];
  let m, order = 1;
  while ((m = re.exec(raw)) !== null) {
    if (m[1].toLowerCase() !== m[3].toLowerCase())
      return { blocks: [], isValid: false, error: 'Mismatched tags' };
    blocks.push({ persona: m[1].toLowerCase(), content: m[2].trim(), order: order++ });
  }
  if (blocks.length < 2) return { blocks: [], isValid: false, error: 'Too few blocks' };
  return { blocks, isValid: true };
}

export function parseWeeklyRoundup(raw: string): ParsedWeeklyRoundup {
  const re = /\[FIXTURE:\s*([^\]]+)\]([\s\S]*?)\[\/FIXTURE\]/gi;
  const fixtures: FixtureDialogue[] = [];
  let m;
  while ((m = re.exec(raw)) !== null) {
    const parsed = parseDialogueContent(m[2]);
    if (!parsed.isValid) return { fixtures: [], isValid: false, error: parsed.error };
    fixtures.push({ matchLabel: m[1].trim(), blocks: parsed.blocks });
  }
  if (!fixtures.length) {
    // Single fixture fallback (no [FIXTURE:] wrapper)
    const parsed = parseDialogueContent(raw);
    if (parsed.isValid) return { fixtures: [{ matchLabel: '', blocks: parsed.blocks }], isValid: true };
    return { fixtures: [], isValid: false, error: 'No fixture blocks found' };
  }
  return { fixtures, isValid: true };
}

// Extended signature
export async function generatePost(
  tenant: IBlogTenant,
  title: string,
  personaTag?: string,
  fixtures?: FixtureEntry[]   // NEW — weekly roundup
): Promise<GeneratedPost>

// Route selection logic
const isRoundup = !!fixtures?.length;

const systemPrompt = isRoundup
  ? buildCombinedSystem(tenant)      // both personas in system
  : personaTag
    ? tenant.blog_persona_prompts?.get(personaTag)  // single persona
    : undefined;                       // fallback to blog_tone

const userMessage = isRoundup
  ? buildRoundupMessage(tenant, title, fixtures)
  : personaTag
    ? buildPersonaMessage(tenant, title, personaTag)
    : buildDefaultMessage(tenant, title);

const response = await anthropic.messages.create({
  model: 'claude-opus-4-6',
  max_tokens: isRoundup ? 5000 : 2000,
  ...(systemPrompt && { system: systemPrompt }),
  messages: [{ role: 'user', content: userMessage }]
});

// buildCombinedSystem — both personas in one system prompt
function buildCombinedSystem(tenant: IBlogTenant): string {
  const kwagga = tenant.blog_persona_prompts?.get('kwagga') ?? '';
  const marcus = tenant.blog_persona_prompts?.get('marcus') ?? '';
  return `You write as TWO alternating personas.

KWAGGA (writes [KWAGGA] blocks):
${kwagga}

MARCUS (writes [MARCUS] blocks):
${marcus}

Switch fully into each persona per block. Genuine disagreement is expected.
Marcus acknowledges Kwagga's point before building his own angle.`;
}

// buildRoundupMessage — fixture list drives structure
function buildRoundupMessage(tenant, title, fixtures): string {
  const list = fixtures.map((f, i) => `${i+1}. ${f.matchLabel}`).join('\n');
  return `Write a weekly rugby betting dialogue: "${title}"

Fixtures:
${list}

For EACH fixture use this exact structure:

[FIXTURE: {matchLabel}]
[KWAGGA]
Kwagga's analysis (150-180 words). Ends with: Bet at [Bookmaker] on [selection].
[/KWAGGA]
[MARCUS]
Marcus's response (150-180 words). References Kwagga by name.
Ends with: Bet at [Bookmaker] on [selection].
[/MARCUS]
[/FIXTURE]

Rules:
- Different bookmakers across fixtures where possible
- Kwagga: set piece, conditions, provincial lens
- Marcus: defensive structure, market inefficiency lens
- They may disagree — that is the point
- Nothing outside the delimiters

JSON frontmatter first:
{ "seo_title": "...", "seo_description": "...", "excerpt": "...",
  "categories": ["Fixture Previews"],
  "tags": ["kwagga","marcus","urc","fixture-preview","weekly-roundup"] }`;
}

22 Angular — MarkdownPipe
ui-content lib
libs/ui/content/src/lib/pipes/markdown.pipe.ts
import { Pipe, PipeTransform } from '@angular/core';
import { DomSanitizer, SafeHtml } from '@angular/platform-browser';
import { marked } from 'marked';

@Pipe({ name: 'markdown', standalone: true })
export class MarkdownPipe implements PipeTransform {
  constructor(private sanitizer: DomSanitizer) {}
  transform(value: string | undefined): SafeHtml {
    if (!value) return '';
    return this.sanitizer.bypassSecurityTrustHtml(marked(value) as string);
  }
}
